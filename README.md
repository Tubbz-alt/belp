# Belp

A simple *B*oolean *E*xpression *L*exer and *P*arser

## Installation

The package can be installed by adding `belp` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:belp, "~> 0.1.0"}
  ]
end
```
